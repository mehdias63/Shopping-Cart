# 🛒 Shopping Cart App

A simple shopping cart web application built with HTML, CSS, and
JavaScript. This project simulates the core functionality of an online
store, allowing users to browse products, add them to a shopping cart,
manage quantities, and view the total order price dynamically.

## 🚀 Features

- Display a list of products
- Add products to the shopping cart
- Increase or decrease product quantity
- Remove products from the cart
- Dynamic cart updates without page refresh
- Real-time total price calculation
- User-friendly shopping experience

## 📖 About The Project

This project is a web application that simulates the basic
functionality of an online shopping store. Users can browse products
and add them to their shopping cart. Every selected product is
automatically added to the cart, where users can review and manage
their chosen items.

Inside the shopping cart page, users can increase or decrease the
quantity of each product, remove products completely, and view
individual product prices. The application also calculates and updates
the total order price dynamically whenever changes are made to the
cart.

The main focus of this project is managing cart state, updating data
dynamically, and providing a smooth user experience. It was developed
to practice essential e-commerce concepts such as Add to Cart,
quantity management, item removal, and real-time price calculation.
